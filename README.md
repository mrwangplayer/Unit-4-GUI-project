# Unit-4-GUI-project

Project introduction / overview
Rhytm game named PARTY

What the project does
This project is an arrow rhytm game where you need to press arrows at the right moment to get score. Having different difficulties.

How to run the program
Use Visual Studio Code and run the main file.

Project goals or purpose
Goals for this porject was to understand how to implement java swing to create simple game versions and maybe give somebody a setup for more advanced game.

Installation or setup instructions
In Visual Studio Code add each file into one project folder with folder for images inside project folder.

Any additional notes or documentation links
Note: Song Selection should have been added, and arrows would spawn with rhytm BPM but that wasn't created properly so it was deleted and only difficulty stayed.
You can change difficulty by changing spawn rate of arrows  adn speed for each difficulty in difficulty java file.
